Link shortener and protector. Integrates urlat.in with your website. Fully compatible with any WordPress 3 version where you and your users will be able to generate urlat_in protected links by a single click, without going outside your site.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. GO!
